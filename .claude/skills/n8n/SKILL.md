# n8n Skill

## Objetivo
Criar, depurar e otimizar workflows no n8n. Conhece a fundo a API interna, nodes customizados e padrões de automação.

## Quando usar
- Criar automações entre sistemas sem código
- Depurar workflows quebrados
- Construir custom nodes para integrações não nativas
- Migrar automações do Zapier/Make para n8n self-hosted

## Como executar
1. Entenda o objetivo da automação: trigger, transformações e destino dos dados
2. Leia `workflow-reference.md` para padrões de estrutura
3. Leia `api-reference.md` se precisar chamar a API do n8n programaticamente
4. Leia `custom-nodes-reference.md` para criar nodes além dos nativos
5. Construa o workflow em etapas: trigger > transform > output
6. Valide com dados reais antes de ativar

## Output esperado
- JSON exportável do workflow pronto para importar no n8n
- Documentação de cada node e sua função
- Instruções de configuração de credenciais necessárias
