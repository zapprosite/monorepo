# /comecar-aqui — Para Começar

## Quando Usar
- Primeira vez no projeto
- Novo no time
- Não sabe por onde começar

## 3 Passos Iniciais

### Passo 1: Entender Contexto
```
/onde-estamos
```
Mostra:
- Em qual SPEC você está
- Status do sistema
- O que está pendente

### Passo 2: Pedir Orientação
```
/tutor
```
Sistema vai:
- Detectar sua situação
- Sugerir próximo passo
- Explicar por quê

### Passo 3: Executar
Siga as sugestões do `/tutor`

## Mapa de Começo

```
🎯 NOVO AQUÍ
     ↓
/onde-estamos ← mostra onde está
     ↓
/tutor ← orientação personalizada
     ↓
[execute recommended command]
```

## Comandos Essenciais

| Comando | Quando |
|---------|--------|
| `/tutor` | Preciso de ajuda |
| `/onde-estamos` | Quero entender contexto |
| `/proximo-passo` | Terminei algo, e agora? |
| `/ajuda-workflow` | Quero entender workflow |

## Não Tenha Medo

O sistema é meant para ajudar.
- `/tutor` não executa nada sem sua aprovação
- Você sempre pode perguntar "por que?"
- Melhor pedir ajuda do que ficar travado
