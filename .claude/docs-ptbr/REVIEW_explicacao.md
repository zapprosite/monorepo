# /review — Code Review

## O que é
O comando `/review` faz análise de qualidade do código antes de merge.

## Quando usar
- Antes de fazer merge de PR
- Quando quer feedback sobre qualidade
- Quando quer identificar problemas potenciais

## O que verifica
- Padrões de código
- Segurança (vulnerabilidades)
- Performance
- Legibilidade
- Boas práticas

## Tipos de Review
- `/review` — review geral
- `/code-review` — análise mais profunda
- `/security-review` — foco em segurança

## Fluxo
```
Implementar → /review → Corrigir issues → Merge
```
