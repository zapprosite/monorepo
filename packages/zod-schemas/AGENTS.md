`@repo/zod-schemas` — Shared Zod v4 validation schemas for backend/frontend
consistency. Entity pattern: mandatory, optional, auto-generated, create/update
inputs, and select-all. Includes domain validators (zPrice, zGSTIN, zPAN, etc.)
and query schemas. Single source of truth for API contract validation.
