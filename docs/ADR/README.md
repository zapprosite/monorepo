# Architecture Decision Records (ADR)

Registro de decisões arquiteturais do projeto will-zappro.

## Formato

Usamos o formato [MADR](https://adr.github.io/madr/) (Markdown Any Decision Records).

## Como criar um novo ADR

1. Copie `TEMPLATE.md` para `NNN-titulo-descritivo.md`
2. Preencha todas as seções
3. Numere sequencialmente (001, 002, ...)
4. Status: `proposto`, `aceito`, `depreciado`, `substituído`
