Focuses on creating stunning, premium, and highly interactive user interfaces.\n---\n\n# /ui-ux-pro-max - Premium Interface Design\n\n$ARGUMENTS\n\n---\n\n## Core Philosophy\n\n\"MVP is an insult. Premium is the baseline. WOW is the goal.\"\n\n---\n\n## workflow\n\n1.  **Analyze Request**: Understand the user's intent and visual requirements.\n2.  **Define Aesthetics**: Select a modern, harmonious color palette and typography.\n3.  **Structure Semantic HTML**: Build the foundation with accessible and SEO-friendly markup.\n4.  **Styling (CSS)**: Use Vanilla CSS to create custom, flexible, and high-performance styles.\n5.  **Interactive Elements**: Add hover effects, micro-animations, and transitions.\n6.  **Responsive Layout**: Ensure the design looks great on all device sizes.\n7.  **Optimize**: Review for performance, accessibility, and visual excellence.\n\n---\n\n## Design Standards\n\n- **Color Palettes**: Avoid generic colors. Use curated HSL colors or sleek dark modes.\n- **Typography**: Use modern Google Fonts (Inter, Roboto, Outfit) instead of browser defaults.\n- **Gradients & Shadows**: Use smooth gradients and subtle box-shadows for depth.\n- **Micro-animations**: Implement subtle CSS transitions for enhanced UX.\n- **No Placeholders**: Generate real images using the `generate_image` tool.\n\n---\n\n## Verification\n\n- [ ] Is it visually stunning?\n- [ ] Are all interactive elements responsive?\n- [ ] Does it use semantic HTML?\n- [ ] Is color contrast sufficient (WCAG)?\n- [ ] Does it wow the user?\n
