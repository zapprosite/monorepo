# Feature Specifications

This directory contains feature specifications (SPECs) that define what to build before coding begins.

## Directory Structure

- `SPEC-*.md` - Individual feature specifications
- `SPEC-TEMPLATE.md` - Template for new specifications

## Workflow

1. Create SPEC using spec-driven development workflow
2. Review and approve before implementation
3. Track in `tasks/pipeline.json`
